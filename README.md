# output 使用導覽

## 目前保留檔案（已去重）
- `index.html`（期中考練習主頁）
  - 一頁整合：**範圍・重點・分數**、**第一／二／三章段落**、**選擇題**、**論述問答**（30 題；每題分 **簡答要點** 與 **詳細論述指引**，後者由 **`sa-data.js`** 載入，請與 `index.html` 同資料夾；可執行 `node build_sa_guides.cjs` 重新產生）。請用瀏覽器開啟 `index.html`。
- `知識管理(分數計算及範圍).md`
  - 主檔：考點、上課內容、注意事項、分數計算、考試範圍。
- `期中考_可能題庫_選擇與問答.md`
  - 練習檔：選擇題 40 題與論述問答 30 題（段落式，非簡答）。
- `知識管理1_段落整理.md`
  - 第一章段落版（補脈絡用）。
- `知識管理2_段落整理.md`
  - 第二章段落版（補脈絡用）。
- `知識管理3_段落整理.md`
  - 第三章段落版（理論與模式整理）。

## 建議使用順序
- 第一步：讀 `知識管理(分數計算及範圍).md` 先掌握全局。
- 第二步：用 `index.html` 做互動練習（或刷 `期中考_可能題庫_選擇與問答.md`）。
- 第三步：回看 `知識管理1_段落整理.md`、`知識管理2_段落整理.md`、`知識管理3_段落整理.md` 補例子與脈絡。

## 每日複習建議（30-40分鐘）
- 10分鐘：主檔重點複誦。
- 15分鐘：選擇題10題。
- 10分鐘：論述問答 1 題（用「定義→差異→例子→應用」段落結構，勿只寫一句）。
